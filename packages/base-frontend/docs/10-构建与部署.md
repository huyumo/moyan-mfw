## 10.1 环境配置

### 10.1.1 环境变量

项目使用 `.env` 文件管理环境变量：

```bash
# .env.development
VITE_APP_TITLE=墨焱管理后台（开发）
VITE_API_BASE_URL=http://localhost:3000

# .env.production
VITE_APP_TITLE=墨焱管理后台
VITE_API_BASE_URL=https://api.example.com
```

### 10.1.2 环境变量使用

```typescript
// 在代码中使用
const apiBaseUrl = import.meta.env.VITE_API_BASE_URL;
const appTitle = import.meta.env.VITE_APP_TITLE;
```

### 10.1.3 环境变量规范

| 变量 | 说明 | 开发环境 | 生产环境 |
|------|------|----------|----------|
| `VITE_API_BASE_URL` | API 基础 URL | http://localhost:3000 | 生产 API 地址 |
| `VITE_APP_TITLE` | 应用标题 | 含（开发）后缀 | 正式标题 |

## 10.2 开发环境

### 10.2.1 启动开发服务器

```bash

pnpm install

pnpm run dev
```

开发服务器运行在 `http://localhost:5173`，支持：

## 10.3 生产构建

### 10.3.1 构建命令

```bash
# 生产构建
pnpm run build

# 类型检查
pnpm run typecheck

# 代码检查
pnpm run lint
```

### 10.3.2 构建输出

```
dist/
├── index.html           # HTML 入口
├── assets/
│   ├── index-[hash].js  # JavaScript 打包
│   ├── style-[hash].css # CSS 打包
│   └── images/          # 静态资源
└── vite/                # Vite 内部文件
```

### 10.3.3 构建优化

Vite 自动进行以下优化：
- 代码分割（Code Splitting）
- Tree Shaking（移除未使用代码）
- 资源压缩（Terser + CSSNano）
- 哈希文件名（缓存优化）

### 10.3.4 本地预览生产构建

```bash
# 构建并预览
pnpm run build
pnpm run preview
```

## 10.4 部署

### 10.4.1 部署准备

1. 确保通过所有测试
2. 确保通过 lint 检查
3. 确保构建成功

### 10.4.2 部署流程

```bash
# 1. 构建
pnpm run build

# 2. 验证 dist 目录
ls -la dist/

# 3. 部署到服务器（示例）
scp -r dist/* user@server:/var/www/html/
```

### 10.4.3 Nginx 配置

```nginx
server {
    listen 80;
    server_name example.com;
    root /var/www/html;
    index index.html;

    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # Gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
```

## 10.5 CI/CD

### 10.5.1 GitHub Actions 示例

```yaml
name: Build and Deploy

on:
  push:
    branches: [main]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v3

      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'

      - name: Install pnpm
        uses: pnpm/action-setup@v2
        with:
          version: 8

      - name: Install dependencies
        run: pnpm install

      - name: Lint
        run: pnpm run lint

      - name: Test
        run: pnpm run test:unit

      - name: Build
        run: pnpm run build

      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./dist
```
