## 4.1 Pinia Store 设计

### 4.1.1 Store 架构

本框架使用 **Pinia** 作为官方状态管理库，采用**模块化**设计，将状态按功能领域分离到不同的 Store 中。

**核心 Store：**

| Store | 文件 | 用途 |
|-------|------|------|
| `layout-store` | `store/layout-store.ts` | 布局配置、主题、导航、标签页管理 |

**Store 组织结构：**

```
packages/base-frontend/src/store/
├── layout-store.ts              # 布局 Store 入口（defineStore）
├── layout-store-model.ts        # 状态模型和类型定义
├── layout-store-utils.ts        # 工具函数（序列化、持久化）
├── layout-store-preference-actions.ts  # 布局偏好设置动作
└── layout-store-tab-actions.ts  # 标签页管理动作
```

### 4.1.2 状态分类

布局状态分为以下几类：

| 状态类型 | 说明 | 示例 | 持久化 |
|----------|------|------|--------|
| **布局样式** | 布局模式、主题、暗黑模式 | `styleConfig` | ✅ |
| **导航配置** | 品牌名称、菜单、首页路径 | `navigation` | ❌ |
| **标签页** | 访问过的标签、激活的标签 | `visitedTabs`, `activeTabPath` | ✅ |
| **扩展组件** | 布局扩展、登录扩展 | `layoutExtensions` | ❌ |
| **应用状态** | 当前应用、用户应用列表 | `currentApp`, `userApps` | ❌ |

### 4.1.3 状态定义示例

```typescript
// store/layout-store-model.ts
export interface LayoutState {
  styleConfig: LayoutStyleConfig;      // 布局样式配置
  navigation: AdminNavigationConfig;   // 导航配置
  settingsPanelOpen: boolean;          // 设置面板状态
  themes: ThemeRegistry;               // 主题注册表
  enableThemeSwitch: boolean;          // 主题切换开关
  activeTopMenuKey: string;            // 当前激活顶部菜单键
  visitedTabs: PageTabItem[];          // 访问过的标签页
  activeTabPath: string;               // 当前激活标签页
  layoutExtensions: LayoutExtensionComponents;
  loginExtensions: LoginExtensionComponents;
  currentApp: AppItem | null;
  userApps: AppItem[];
}
```

### 4.1.4 Actions 设计规范

Actions 采用**函数分离**模式，将逻辑提取到独立文件中：

```typescript
// store/layout-store.ts
export const useLayoutStore = defineStore('mfw-base-layout', {
  state: (): LayoutState => { ... },

  getters: {
    activeThemeKey: (state) => { ... },
    activeThemeTokens: (state) => { ... },
  },

  actions: {
    // 偏好设置类动作
    patchStyleConfig(payload, options) {
      patchStyleConfig(this as LayoutPreferenceActionContext, payload, options);
    },

    // 标签页管理类动作
    closeTab(path: string): string {
      return closeTab(this as LayoutPreferenceActionContext, path);
    },
  },
});
```

## 4.2 本地状态 vs 全局状态

### 4.2.1 状态选择原则

| 场景 | 推荐方案 | 说明 |
|------|----------|------|
| 组件内部 UI 状态 | `ref` / `reactive` | 按钮状态、表单输入、显示/隐藏 |
| 父子组件共享 | `props` + `emits` | 单向数据流，事件向上传递 |
| 跨组件/跨页面共享 | Pinia Store | 主题、布局、用户信息、全局配置 |
| 服务端状态缓存 | Pinia Store + 持久化 | API 响应数据、列表缓存 |

### 4.2.2 组件内状态

```vue
<script setup lang="ts">
import { ref, reactive } from 'vue';

// 简单状态 - 使用 ref
const isLoading = ref(false);
const searchText = ref('');

// 对象状态 - 使用 reactive
const form = reactive({
  username: '',
  password: '',
});

// 列表状态
const items = ref<string[]>([]);
</script>
```

### 4.2.3 全局状态

```typescript
// 使用全局 Store
import { useLayoutStore } from 'moyan-mfw-base-frontend';

const layoutStore = useLayoutStore();

// 读取状态
const theme = layoutStore.styleConfig.theme;
const isDark = layoutStore.styleConfig.isDark;

// 调用动作
layoutStore.setTheme('sunset');
layoutStore.toggleSettingsPanel();
```

### 4.2.4 持久化状态

布局偏好设置会自动持久化到 `localStorage`：

| 状态 | 存储键名 | 说明 |
|------|----------|------|
| 样式配置 | `mfw:base-frontend:layout-preferences` | 主题、布局模式、暗黑模式等 |
| 标签页 | 同上 | 访问过的标签、激活的标签 |

**持久化数据结构：**

```typescript
interface LayoutPersistedState {
  styleConfig?: Partial<LayoutStyleConfig>;
  activeTopMenuKey?: string;
  visitedTabs?: PageTabItem[];
  activeTabPath?: string;
}
```

## 4.3 标签页管理

### 4.3.1 标签页状态

标签页状态存储在 `visitedTabs` 数组中，每个标签包含：

```typescript
interface PageTabItem {
  fullPath: string;      // 完整路径
  path: string;          // 路径
  name: string;          // 路由名称
  title: string;         // 标签标题
  icon?: string;         // 图标
  closable?: boolean;    // 是否可关闭
}
```

### 4.3.2 标签操作

| 方法 | 说明 | 返回值 |
|------|------|--------|
| `closeTab(path)` | 关闭指定标签 | 新的激活标签路径 |
| `closeTabsLeft(path)` | 关闭左侧所有标签 | 新的激活标签路径 |
| `closeTabsRight(path)` | 关闭右侧所有标签 | 新的激活标签路径 |
| `closeOtherTabs(path)` | 关闭其他所有标签 | 新的激活标签路径 |
| `closeAllTabs()` | 关闭所有标签 | 首页路径 |
| `setActiveTab(path)` | 设置激活标签 | - |

### 4.3.3 使用示例

```typescript
import { useLayoutStore } from 'moyan-mfw-base-frontend';

const layoutStore = useLayoutStore();

// 关闭当前标签并返回上一个
const newPath = layoutStore.closeTab(route.fullPath);
router.push(newPath);

// 关闭其他标签
layoutStore.closeOtherTabs(route.fullPath);

// 设置激活标签
layoutStore.setActiveTab('/dashboard');
```

## 4.4 主题管理

### 4.4.1 主题注册

```typescript
import { useLayoutStore } from 'moyan-mfw-base-frontend';

const layoutStore = useLayoutStore();

// 注册自定义主题
layoutStore.registerThemes({
  'custom-theme': {
    label: '自定义主题',
    tokens: {
      light: {
        primary: '#1890ff',
        success: '#52c41a',
        warning: '#faad14',
        error: '#f5222d',
      },
      dark: {
        primary: '#177ddc',
        success: '#389e0d',
        warning: '#d89614',
        error: '#a61d1d',
      },
    },
  },
});
```

### 4.4.2 主题切换

```typescript
// 切换主题
layoutStore.setTheme('sunset');

// 切换暗黑模式
layoutStore.setDarkMode(true);

// 读取当前主题令牌
const tokens = layoutStore.activeThemeTokens;
```
