## 归档说明

- **归档原因**: 文档内容已过时、项目已结束或被新文档替代

- **查阅建议**: 如需了解最新规范，请查阅 `docs/03-框架规范/` 等活跃目录

## 归档流程

```
1. 识别过时文档
   ↓
2. 标记 status: deprecated
   ↓
3. 添加废弃说明和替代文档链接
   ↓
4. 保留 30 天观察期
   ↓
5. 移动至 archived 目录
   ↓
6. 更新本索引
```
