## 与框架规范的关系

```
项目实施文档                    框架规范文档
     ↓                              ↓
业务-specific 实现指南    ←→    通用开发规范
     ↓                              ↓
  具体业务模块                   编码标准
```

- **项目实施文档**：针对当前业务项目的具体实现指南
- **框架规范文档**：前端/后端框架开发的通用规范

开发时应同时参考项目实施文档和框架规范文档。

---

**维护说明**：
- 项目实施文档应随着项目迭代持续更新
- 新增业务功能应及时补充开发文档
- API 变更应同步更新 API 开发文档
