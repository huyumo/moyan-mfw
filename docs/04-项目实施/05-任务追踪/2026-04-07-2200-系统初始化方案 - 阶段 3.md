### 4. 添加 readRoutes 函数

**文件修改**：`packages/base-frontend/src/router/menu-tree.ts`

```typescript
/**
 * 读取路由配置并转换为菜单格式（用于安装向导预览）
 * @returns 菜单树形结构
 */
export function readRoutes(): SideMenuItem[] {
  // 从 baseAdminRoutes 中提取菜单树
  const mainRoute = baseAdminRoutes.find(route => route.path === '/')
  if (!mainRoute || !mainRoute.children) {
    return []
  }

  return createMenuTreeFromRoutes(mainRoute.children, { parentPath: '' })
}
```

## 安装流程

```
1. 用户访问前端首页
   ↓
2. 路由守卫检测到未登录，重定向到 /login
   ↓
3. 用户手动访问 /install（或系统检测到未初始化自动跳转）
   ↓
4. 显示安装向导页面
   ↓
5. 步骤 1：设置 admin 管理员密码
   - 输入密码（8-32 位，包含字母和数字）
   - 确认密码
   - 点击"下一步"
   ↓
6. 步骤 2：预览路由并确认
   - 显示前端路由树形结构
   - 点击"上一步"返回修改
   - 点击"开始初始化"
   ↓
7. 调用 POST /api/install/init
   - 后端执行种子数据创建
   - 同步前端路由到权限树
   - 返回初始化结果
   ↓
8. 初始化成功，跳转到 /login
   ↓
9. 用户使用 admin 账号和新密码登录
```
