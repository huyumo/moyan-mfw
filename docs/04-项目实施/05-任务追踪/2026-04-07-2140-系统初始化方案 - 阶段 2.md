## 完成内容

### 1. 创建 InstallService 服务

**文件位置**：`packages/base-backend/src/modules/sys/install/install.service.ts`

**核心方法**：
- `isInitialized()` - 检查系统是否已初始化（通过检查 AppType 表是否有记录）
- `initialize()` - 执行初始化流程（执行种子数据 + 同步前端路由）

**代码实现**：
```typescript
@Injectable()
export class InstallService {
  constructor(
    @InjectRepository(AppType)
    private appTypeRepository: Repository<AppType>,
    private dataSource: DataSource,
    private permissionService: PermissionService,
  ) {}

  async isInitialized(): Promise<boolean> {
    const count = await this.appTypeRepository.count();
    return count > 0;
  }

  async initialize(initData: {
    adminPassword: string;
    routes: RouteNodeDto[];
  }): Promise<InitResponseDto> {
    if (await this.isInitialized()) {
      throw new ConflictException('系统已初始化，无法重复执行');
    }

    await runSeeds(this.dataSource);
    await this.permissionService.syncPermissions(initData.routes);

    // 返回初始化结果
    return { appTypeId, appId, adminUserId, message: '初始化成功' };
  }
}
```

### 2. 创建 InstallController 控制器

**文件位置**：`packages/base-backend/src/modules/sys/install/install.controller.ts`

**API 接口**：
| 接口 | 方法 | 说明 | 认证 |
|------|------|------|------|
| `/api/install/status` | GET | 检查系统是否已初始化 | 公开 |
| `/api/install/init` | POST | 执行系统初始化 | 公开 |

**代码实现**：
```typescript
@ApiTags('系统初始化')
@Controller('install')
export class InstallController {
  constructor(private readonly installService: InstallService) {}

  @Get('status')
  @Public()
  async getStatus(): Promise<InitStatusResponseDto> {
    const initialized = await this.installService.isInitialized();
    return { initialized };
  }

  @Post('init')
  @Public()
  async initialize(@Body() initData: InitRequestDto): Promise<InitResponseDto> {
    return this.installService.initialize(initData);
  }
}
```

### 3. 创建 DTO 类型定义

#### InitRequestDto
**文件位置**：`packages/base-backend/src/modules/sys/install/dto/init-request.dto.ts`

```typescript
export class InitRequestDto {
  @IsString()
  @MinLength(8)
  @MaxLength(32)
  adminPassword: string;

  @IsArray()
  @ValidateNested({ each: true })
  @Type(() => RouteNodeDto)
  routes: RouteNodeDto[];
}

export class RouteNodeDto {
  @IsString()
  path: string;

  @IsString()
  name: string;

  @IsArray()
  @ValidateNested({ each: true })
  children?: RouteNodeDto[];
}
```

## 关键决策

- **初始化检测逻辑**：使用 `AppType` 表是否有记录来判断系统是否已初始化，简单可靠
- **公开接口**：初始化接口必须是公开的，否则无法在首次访问时调用
- **事务处理**：种子数据自己管理事务，`initialize()` 方法不再额外包装事务
- **路由同步**：初始化时自动调用 `permissionService.syncPermissions()` 同步前端路由到权限树
