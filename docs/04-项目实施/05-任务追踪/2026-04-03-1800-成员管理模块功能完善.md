## 关键经验

### moyan-api 使用规范
```
1. 后端修改 Swagger 注解
2. 提交代码
3. 启动后端服务
4. 运行 pnpm run api:build
5. 前端自动更新 apis/
```

### API 调用参数规范
```typescript
// query: URL 路径参数（如 /apps/{appId}/members 中的 appId）
// params: 请求体参数（POST/PUT 的 body）

new ApiMemberAddMember({
  query: { appId: props.data!.appId },  // 路径参数
  params: { userId: form.userId },       // 请求体
});
```
