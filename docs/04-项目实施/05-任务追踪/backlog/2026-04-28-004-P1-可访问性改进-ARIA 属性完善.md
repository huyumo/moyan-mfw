### 2. 按钮组件 ARIA 改进 (25min)

#### 2.1 为图标按钮添加 aria-label
- [ ] **2.1.1** 打开 `packages/base-frontend/src/components/` 目录
- [ ] **2.1.2** 找到所有只有图标没有文字的按钮
- [ ] **2.1.3** 为每个按钮添加 `aria-label` 属性
- [ ] **2.1.4** 示例：`<el-button aria-label="删除"><Delete /></el-button>`
- [ ] **2.1.5** 记录修改的组件列表

#### 2.2 为操作按钮组添加 role
- [ ] **2.2.1** 找到所有按钮组容器
- [ ] **2.2.2** 添加 `role="group"` 属性
- [ ] **2.2.3** 添加 `aria-label` 描述组功能
- [ ] **2.2.4** 示例：`<div role="group" aria-label="操作按钮">`

#### 2.3 为切换按钮添加 aria-pressed
- [ ] **2.3.1** 找到所有切换状态按钮（如展开/收起）
- [ ] **2.3.2** 添加 `:aria-pressed="isActive"` 属性
- [ ] **2.3.3** 验证按钮状态变化时 aria-pressed 同步更新

**完成标志**: 所有按钮组件通过 ARIA 检查

---

### 3. 表单组件 ARIA 改进 (35min)

#### 3.1 为表单项添加 aria-describedby
- [ ] **3.1.1** 打开 `MfwFormCard.vue` 组件
- [ ] **3.1.2** 为每个 `<el-form-item>` 添加 `:id="`field-${index}`"`
- [ ] **3.1.3** 在错误提示元素上添加 `role="alert"` 和 `:id="`error-${index}`"`
- [ ] **3.1.4** 在输入框上添加 `:aria-describedby="`error-${index}`"`

#### 3.2 为必填字段添加 aria-required
- [ ] **3.2.1** 扫描所有表单字段
- [ ] **3.2.2** 识别必填字段（有 rules 且 required: true）
- [ ] **3.2.3** 添加 `aria-required="true"` 属性
- [ ] **3.2.4** 在 label 上添加视觉标识（*）

#### 3.3 为表单错误添加 aria-live
- [ ] **3.3.1** 找到所有错误提示元素
- [ ] **3.3.2** 添加 `role="alert"` 和 `aria-live="polite"`
- [ ] **3.3.3** 验证错误出现时屏幕阅读器可通知

#### 3.4 为输入框添加 aria-invalid
- [ ] **3.4.1** 找到所有输入框组件
- [ ] **3.4.2** 添加 `:aria-invalid="hasError"` 属性
- [ ] **3.4.3** 验证验证失败时属性值为 "true"

**完成标志**: 所有表单组件符合 WCAG 2.1 AA 标准

### 4. 表格组件 ARIA 改进 (25min)

#### 4.1 为表格添加 role 属性
- [ ] **4.1.1** 打开 `MfwTableList.vue` 组件
- [ ] **4.1.2** 确认 `<el-table>` 有 `role="grid"` 属性
- [ ] **4.1.3** 如无，手动添加

#### 4.2 为表格添加 aria-label
- [ ] **4.2.1** 在 `<el-table>` 上添加 `aria-label` 属性
- [ ] **4.2.2** 值描述表格内容，如 `aria-label="用户列表"`

#### 4.3 为排序列添加 aria-sort
- [ ] **4.3.1** 找到所有可排序的列头
- [ ] **4.3.2** 添加 `:aria-sort="sortOrder"` 属性
- [ ] **4.3.3** 值可以是 "ascending"、"descending"、"none"

#### 4.4 为行选择添加 aria-selected
- [ ] **4.4.1** 找到所有可选择的行
- [ ] **4.4.2** 添加 `:aria-selected="isSelected"` 属性

**完成标志**: 表格组件支持屏幕阅读器导航
