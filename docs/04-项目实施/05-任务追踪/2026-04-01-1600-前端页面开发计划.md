## 详细任务分解

### 任务 1：应用类型管理页面（P1）

**需求文档**：`docs/01-业务需求/01-基础设施/05-页面设计/应用类型管理页面.md`

**核心功能**：
- [ ] 应用类型列表展示（开发者模式专属）
- [ ] 应用类型编辑（typeName、icon、typeDesc）
- [ ] 权限池配置面板（PC 权限树、普通权限列表、permissionValue 配置）
- [ ] 内置角色管理（增删改、权限分配）
- [ ] RolePermissionPanel 组件（复用）

**技术要点**：
- 开发者模式鉴权（后端路由鉴权）
- 权限池按 appTypeId 隔离
- permissionValue 位运算配置

**验收标准**：
- 非开发者用户无法访问
- 权限池配置正确保存
- 内置角色权限分配正确

### 任务 2：应用实例管理页面（P1）

**需求文档**：`docs/01-业务需求/01-基础设施/05-页面设计/应用实例管理页面.md`

**核心功能**：
- [ ] 应用实例列表展示（支持按类型筛选）
- [ ] 应用实例创建（选择类型、绑定拥有者）
- [ ] 应用实例编辑（信息修改、拥有者变更）
- [ ] 应用实例删除
- [ ] 应用详情展示（含拥有者信息）
- [ ] 成员管理入口链接

**技术要点**：
- 拥有者搜索（手机号查询用户）
- 拥有者变更事务处理（后端）
- 内置应用实例处理（system-instance）

**验收标准**：
- 应用实例必须始终有拥有者
- 拥有者变更自动绑定拥有者角色
- 成员管理入口正确跳转

### 任务 3：角色管理页面（P1）

**需求文档**：`docs/01-业务需求/01-基础设施/05-页面设计/角色管理页面.md`

**核心功能**：
- [ ] 角色列表展示（内置角色 + 应用级角色）
- [ ] 内置角色只读展示（禁用编辑/删除/权限分配）
- [ ] 应用级角色管理（增删改）
- [ ] 权限分配面板（RolePermissionPanel 组件）
- [ ] 角色删除级联处理

**技术要点**：
- 内置角色与应用级角色区分
- 权限分配从应用类型权限池选择
- permissionValue 配置支持

**验收标准**：
- 内置角色操作按钮正确禁用
- 应用级角色权限分配正确
- 角色删除级联清理权限关联

### 任务 4：成员管理页面（P1）

**需求文档**：`docs/01-业务需求/01-基础设施/05-页面设计/成员管理页面.md`

**核心功能**：
- [ ] 成员列表展示（不包括拥有者）
- [ ] 添加成员（搜索用户、确认添加）
- [ ] 移除成员
- [ ] 角色分配面板（内置角色 + 应用级角色）
- [ ] 多角色支持

**技术要点**：

**验收标准**：
- 成员列表正确显示
- 角色分配正确保存
- 多角色权限计算正确
