# 墨言权限管理系统 - 实施方案

> 版本：1.0.0
> 制定日期：2026-03-28
> 执行周期：预计 6 周

---

## 1. 实施目标

### 1.1 总体目标

完成墨言权限管理系统的核心功能开发，实现：
- 应用类型中心模式的权限管理体系
- 可视化的权限池配置和管理
- 细粒度的位运算权限控制
- 完整的前后端权限验证

### 1.2 阶段目标

| 阶段 | 周期 | 目标 |
|------|------|------|
| 第一阶段 | 第 1-2 周 | 后端核心 API + 应用类型管理页面 |
| 第二阶段 | 第 3-4 周 | 角色/权限管理页面 + 认证登录 |
| 第三阶段 | 第 5-6 周 | 成员管理 + 审计日志 + 联调测试 |

---

## 3. 第一阶段：基础核心（第 1-2 周）

### 3.1 后端任务

#### 3.1.1 数据库初始化
- [ ] 创建 10 张核心表
- [ ] 初始化索引和约束
- [ ] 种子数据（超级管理员、系统应用类型）

#### 3.1.2 应用类型模块
- [ ] `GET /api/app-types` - 获取应用类型列表
- [ ] `POST /api/app-types` - 创建应用类型
- [ ] `PUT /api/app-types/:id` - 更新应用类型
- [ ] `DELETE /api/app-types/:id` - 删除应用类型
- [ ] `GET /api/app-types/:id/permissions` - 获取权限池
- [ ] `PUT /api/app-types/:id/permissions` - 配置权限池
- [ ] `GET /api/app-types/:id/builtin-roles` - 获取内置角色列表
- [ ] `POST /api/app-types/:id/builtin-roles` - 创建内置角色

#### 3.1.3 应用实例模块
- [ ] `GET /api/apps` - 获取应用实例列表
- [ ] `POST /api/apps` - 创建应用实例
- [ ] `PUT /api/apps/:id` - 更新应用实例
- [ ] `PUT /api/apps/:id/owner` - 变更拥有者
- [ ] `GET /api/apps/:id/members` - 获取成员列表

#### 3.1.4 权限计算服务
- [ ] 权限并集计算（位运算 OR）
- [ ] 权限池约束验证
- [ ] Redis 缓存（30 分钟过期）

## 6. 开发规范

### 6.1 代码规范

- 使用 ESLint + Prettier 格式化代码
- TypeScript 严格模式
- Vue 3 Composition API + `<script setup>`
- 组件命名：PascalCase（如 `AppTypeListPage.vue`）
- 文件注释：每个文件顶部添加 `@fileoverview`

### 6.2 Git 规范

```bash
# 提交信息格式
<type>: <subject>

# type 类型
- feat: 新功能
- fix: Bug 修复
- docs: 文档更新
- chore: 构建/工具配置
- refactor: 代码重构
- test: 测试相关
```

### 6.3 分支策略

```
main              # 主分支，保护
├── develop       # 开发分支
│   ├── feature/app-type-mgmt   # 功能分支
│   ├── feature/role-mgmt
│   └── feature/auth
```

### 6.4 目录规范

```
frontend/src/
├── business/
│   ├── views/           # 页面视图
│   ├── components/      # 业务组件
│   ├── api/             # API 调用
│   ├── composables/     # 组合式函数
│   └── types/           # TypeScript 类型
```

## 8. 风险管理

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 后端 API 延期 | 中 | 高 | 优先开发核心接口，前端使用 Mock |
| 位运算权限计算复杂 | 中 | 中 | 编写工具函数，增加单元测试 |
| 前后端联调问题 | 高 | 高 | 提前对齐接口定义，使用 Swagger |
| 性能问题 | 低 | 中 | 权限计算加缓存，数据库加索引 |
