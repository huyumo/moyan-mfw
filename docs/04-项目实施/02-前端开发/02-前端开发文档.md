## 1. 前端架构概述

### 1.1 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                    业务前端应用                          │
│                  (frontend/)                             │
│  ┌──────────────────────────────────────────────────┐   │
│  │  business/                                        │   │
│  │  ├── views/          (业务视图组件)                │   │
│  │  ├── components/     (业务组件)                    │   │
│  │  ├── api/            (API 调用层)                   │   │

│  │  moyan-mfw-base-frontend  (基础框架包)            │   │
│  │  ├── layouts/    (布局系统)                       │   │
```

### 3.2 Composables 模式

```typescript
// frontend/src/business/views/order/composables/use-order-center.ts
import { ref, computed } from 'vue';
import { orderApi } from '../api';
import type { OrderMetrics, ActivityItem } from '../types';

/**
 * 订单中心页面逻辑
 */
export function useOrderCenter() {
  const loading = ref(false);
  const summaryMetrics = ref<OrderMetrics | null>(null);
  const activityItems = ref<ActivityItem[]>([]);

  // 加载数据
  const loadData = async () => {
    loading.value = true;
    try {
      const [metrics, activities] = await Promise.all([
        orderApi.getSummaryMetrics(),
        orderApi.getActivityTimeline(),
      ]);
      summaryMetrics.value = metrics;
      activityItems.value = activities;
    } finally {
      loading.value = false;
    }
  };

  // 刷新数据
  const refresh = async () => {
    await loadData();
  };

  return {
    loading,
    summaryMetrics,
    activityItems,
    refresh,
  };
}
```

### 3.3 API 调用层

```typescript
// frontend/src/business/views/order/api/order-api.ts
import axios from 'axios';
import type { OrderMetrics, ActivityItem } from '../types';

const api = axios.create({
  baseURL: '/api',
  timeout: 10000,
});

export const orderApi = {
  /**
   * 获取订单概览指标
   */
  async getSummaryMetrics(): Promise<OrderMetrics> {
    const { data } = await api.get<OrderMetrics>('/orders/metrics');
    return data;
  },

  /**
   * 获取订单活动时间线
   */
  async getActivityTimeline(): Promise<ActivityItem[]> {
    const { data } = await api.get<ActivityItem[]>('/orders/activity');
    return data;
  },
};
```

```typescript
export interface OrderMetrics {
```

## 4. 组件开发规范

### 4.1 组件结构

```vue
<!--
/**
 * @fileoverview 组件说明
 */
-->
<template>
  <div class="component-name">
    <!-- 组件内容 -->
  </div>
</template>

<script setup lang="ts">
import { computed, defineProps, defineEmits } from 'vue';

// Props 定义
interface Props {
  modelValue?: string;
  disabled?: boolean;
}
const props = withDefaults(defineProps<Props>(), {
  modelValue: '',
  disabled: false,
});

// Emits 定义
interface Emits {
  (e: 'update:modelValue', value: string): void;
  (e: 'change', value: string): void;
}
const emits = defineEmits<Emits>();

// 组件逻辑
</script>

<style scoped lang="scss">
.component-name {
  // 组件样式
}
</style>
```

### 4.2 Props 规范

```typescript
// ✅ 推荐：使用 interface + withDefaults
interface Props {
  modelValue?: string;
  disabled?: boolean;
  options?: Option[];
}
const props = withDefaults(defineProps<Props>(), {
  modelValue: '',
  disabled: false,
  options: () => [],
});

// ❌ 不推荐：内联定义
const props = defineProps({
  modelValue: {
    type: String,
    default: '',
  },
});
```

### 4.3 组件命名

| 类型 | 命名方式 | 示例 |
|------|----------|------|
| 页面组件 | `XxxPage.vue` | `OrderCenterPage.vue` |
| 业务组件 | `XxxCard.vue`, `XxxList.vue` | `OrderSummaryCard.vue` |
| 基础组件 | `BaseXxx.vue` | `BaseSelect.vue` |
| 布局组件 | `XxxPanel.vue` | `HeaderPanel.vue` |

## 7. 开发流程

### 7.1 新建页面流程

1. **创建路由配置**
   ```typescript
   // frontend/src/business/routes.ts
   {
     path: 'new-page',
     name: 'NewPage',
     component: () => import('./views/new-page/NewPage.vue'),
     meta: {
       title: '新页面',
       menuLabel: '新页面',
       menuIcon: 'Document',
       menuOrder: 1,
     },
   }
   ```

2. **创建页面文件**
   ```bash
   frontend/src/business/views/new-page/
   ├── NewPage.vue              # 主页面
   ├── components/              # 页面级组件
   ├── composables/             # 组合式函数
   ├── api/                     # API 调用
   └── types/                   # 类型定义
   ```

3. **实现页面逻辑**
   ```vue
   <script setup lang="ts">
   import { useNewPage } from './composables/use-new-page';
   const { loading, data } = useNewPage();
   </script>
   ```

4. **测试验证**
   - 路由是否正确注册
   - 菜单是否正确显示
   - 页面功能是否正常
