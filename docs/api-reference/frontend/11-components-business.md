# 11 · 业务类组件

全部可从 `moyan-mfw-base/frontend` 导入。

---

## `RoleCard`

角色卡片组件，展示角色基本信息。

```typescript
import { RoleCard } from 'moyan-mfw-base/frontend'
```

---

## `RoleForm`

角色表单组件（新增/编辑角色）。

```typescript
import { RoleForm } from 'moyan-mfw-base/frontend'
```

---

## `PermissionTree`

权限树组件，展示和选择权限节点。

```typescript
import { PermissionTree } from 'moyan-mfw-base/frontend'
```

---

## `PermissionManager`

权限管理器组件，完整的权限节点增删改查。

```typescript

```
