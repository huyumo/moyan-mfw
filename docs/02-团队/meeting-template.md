## 会议类型说明

### 每日站会 (Daily Standup)

**时间**: 每个工作日 14:00-14:15
**参与者**: 全体成员
**议程**:
1. 昨日完成情况（每人 1 分钟）
2. 今日计划（每人 1 分钟）
3. 遇到的问题（集中讨论）

### 周例会 (Weekly Meeting)

**时间**: 每周五 14:00-15:00
**参与者**: 全体成员
**议程**:
1. 本周工作总结
2. 下周工作计划
3. 需要协调的问题
4. 其他事项

### 评审会 (Review Meeting)

**时间**: 按需安排
**参与者**: 相关角色 + 审查人员
**议程**:
1. 成果展示
2. 审查意见
3. 改进建议
4. 验收结论

### 回顾会 (Retrospective)

**时间**: 每阶段结束
**参与者**: 全体成员
**议程**:
1. 做得好的地方（继续保持）
2. 需要改进的地方（改进计划）
3. 行动计划（具体措施）
