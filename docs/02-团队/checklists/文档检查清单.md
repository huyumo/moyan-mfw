# 文档工程师角色检查清单

## 文档结构

- [ ] 目录结构是否清晰
- [ ] 章节划分是否合理
- [ ] 导航链接是否正确
- [ ] 是否有 404 死链接

## 内容质量

- [ ] 术语使用是否一致
- [ ] 表述是否清晰无歧义
- [ ] 示例是否准确
- [ ] 图表是否清晰
- [ ] 代码示例是否可运行

## 一致性检查

- [ ] 与代码实现是否一致
- [ ] 与 API 文档是否一致
- [ ] 与数据库设计是否一致
- [ ] 与其他文档是否冲突

## 版本管理

- [ ] 版本号是否更新
- [ ] 变更历史是否记录
- [ ] 废弃内容是否标记
- [ ] 兼容性说明是否清晰

## 可读性

- [ ] 语言是否简洁
- [ ] 是否有拼写/语法错误
- [ ] 格式是否统一
- [ ] 中英文混排是否规范

## SEO 与搜索

- [ ] 标题是否描述清晰
- [ ] 关键词是否合理
- [ ] 搜索是否能找到相关内容
