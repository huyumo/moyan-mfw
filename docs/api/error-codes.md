# API 错误码定义

> 本文档定义所有 API 接口的通用错误码
>
> **版本**: 1.0.0

---

## 错误响应结构

所有 API 错误响应采用统一格式：

```typescript
{
  code: number;                  // 业务错误码
  data: null;                    // 错误时 data 为 null
  message: string;               // 错误描述
}
```

---

## 通用错误码

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 0 | 200 | 成功 | - |
| 40001 | 400 | 请求参数错误 | 检查请求参数格式和必填字段 |
| 40002 | 400 | 参数校验失败 | 检查字段长度、格式等约束 |
| 40101 | 401 | 未授权/登录过期 | 刷新 Token 或重新登录 |
| 40102 | 401 | Token 无效 | 重新登录 |
| 40301 | 403 | 无权限访问 | 联系管理员分配权限 |
| 40302 | 403 | 开发者模式受限 | 需要 isDeveloper = 1 的用户 |
| 40401 | 404 | 资源不存在 | 检查 ID 参数 |
| 40402 | 404 | 应用不存在 | 检查 appId |
| 40403 | 404 | 角色不存在 | 检查 roleId |
| 40404 | 404 | 权限不存在 | 检查 permissionId |
| 40901 | 409 | 资源冲突 | 用户名/手机号已存在 |
| 40902 | 409 | 编码已存在 | permCode/roleCode/appCode 已存在 |
| 40903 | 409 | 手机号已被使用 | 更换手机号或联系客服 |
| 42201 | 422 | 业务规则约束 | 如：拥有者角色不允许删除 |
| 42202 | 422 | 存在关联数据 | 资源被其他数据引用，无法删除 |
| 50001 | 500 | 服务器内部错误 | 联系系统管理员 |

## 各模块特有错误码

### 认证模块 (auth-api.md)

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 40103 | 401 | 用户名或密码错误 | 检查账号密码 |
| 40104 | 401 | 账号已被禁用 | 联系管理员启用账号 |
| 40105 | 401 | refreshToken 已过期 | 重新登录 |
| 42203 | 422 | 原密码错误 | 输入正确的原密码 |

---

### 用户模块 (user-api.md)

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 40904 | 409 | 用户名已存在 | 更换用户名 |
| 40905 | 409 | 手机号已存在 | 更换手机号或直接登录 |
| 40906 | 409 | 用户名或手机号冲突 | username 或 phone 与现有用户重复 |

---

### 应用类型模块 (app-type-api.md)

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 40405 | 404 | 应用类型不存在 | 检查 appTypeId |
| 40907 | 409 | 应用类型编码已存在 | 更换 typeCode |
| 40912 | 409 | 应用类型编码已存在 | 更换 typeCode |
| 42204 | 422 | 系统内置类型不允许删除 | typeCode = 'system' 不可删除 |
| 42205 | 422 | 拥有者角色不允许删除 | isOwner = 1 的角色不可删除 |
| 42206 | 422 | 存在关联应用实例 | 该应用类型下已有应用，无法删除 |

---

### 权限池模块

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 42215 | 422 | 权限池未配置 | 先配置权限池再分配角色权限 |
| 42216 | 422 | pcAction 超出权限池范围 | 角色 pcAction 必须是权限池的子集 |

---

### 角色模块 (role-api.md)

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 40909 | 409 | 角色编码已存在 | 更换 roleCode |
| 42209 | 422 | 拥有者角色不允许删除 | isOwner = 1 的角色不可删除 |
| 42210 | 422 | 角色已绑定用户 | 先解除用户绑定再删除 |

---

### 权限模块 (permission-api.md)

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 40910 | 409 | 权限编码已存在 | 更换 permCode |
| 42211 | 422 | PAGE 节点的父节点必须是 MENU | 检查 parentId 指向的节点类型 |
| 42212 | 422 | 权限已用于权限池 | 先从权限池移除再删除 |

---

### 成员模块 (member-api.md)

| 错误码 | HTTP 状态 | 说明 | 处理建议 |
|--------|----------|------|----------|
| 40911 | 409 | 用户已是成员 | 无需重复添加 |
| 42213 | 422 | 不允许移除拥有者 | 拥有者只能在应用管理中变更 |
| 42214 | 422 | 不允许给自己分配角色 | 成员不能给自己分配角色 |

---

## 错误响应示例

### 参数校验失败

```json
{
  "code": 40002,
  "data": null,
  "message": "参数校验失败：username 长度必须在 2-20 之间"
}
```

### 资源不存在

```json
{
  "code": 40401,
  "data": null,
  "message": "资源不存在：用户 ID 'usr-123' 不存在"
}
```

### 资源冲突

```json
{
  "code": 40904,
  "data": null,
  "message": "用户名已存在：'admin'"
}
```

### 业务规则约束

```json
{
  "code": 42204,
  "data": null,
  "message": "系统内置类型不允许删除：typeCode = 'system'"
}
```

### 未授权

```json
{
  "code": 40101,
  "data": null,
  "message": "Token 已过期，请刷新后重试"
}
```
