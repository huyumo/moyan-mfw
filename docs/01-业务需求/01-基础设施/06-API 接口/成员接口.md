# 成员管理接口

> **相关页面**: [成员管理页面](../05-页面设计/成员管理页面.md)
>
> **数据结构**: [UserEntity](../03-数据库设计/数据库实体设计.md#15-userentity-用户实体), [RoleEntity](../03-数据库设计/数据库实体设计.md#13-roleentity-角色实体)
>
> **复用接口**: [搜索用户](./用户接口.md#1-搜索用户)
>
> **认证**: 所有接口需要在请求头中携带 `Authorization: Bearer <token>`

---

## 接口概览

| 接口 | 方法 | 使用场景 |
|------|------|----------|
| `/api/v1/apps/:appId/members` | GET | 成员列表页加载 |
| `/api/v1/apps/:appId/members` | POST | 添加成员 |
| `/api/v1/apps/:appId/members/:userId` | DELETE | 移除成员 |
| `/api/v1/apps/:appId/members/:userId/roles` | GET | 获取成员角色配置 |
| `/api/v1/apps/:appId/members/:userId/roles` | PUT | 更新成员角色配置 |
| `/api/v1/apps/:appId/available-roles` | GET | 获取可选角色列表 |

---

## 1. 获取成员列表

**接口**: `GET /api/v1/apps/:appId/members`

**使用场景**: 成员管理页面加载，展示成员列表（不包括拥有者）。

**路径参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| appId | string | 是 | 应用 ID |

**返回数据**:

```typescript
{
  code: number;
  data: Array<{
    id: string;                          // 用户应用关联 ID
    userId: string;
    appId: string;
    createAt: string;
    user: {
      id: string;
      username: string;
      nickname?: string;
      phone?: string;
      avatar?: string;
    };
    roles: Array<{
      roleId: string;
      roleName: string;
      roleCode: string;
      isBuiltin: number;
    }>;
  }>;
  message?: string;
}
```

---

## 2. 添加成员

**接口**: `POST /api/v1/apps/:appId/members`

**使用场景**: 点击添加成员按钮，搜索并选择用户后保存。

**路径参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| appId | string | 是 | 应用 ID |

**请求体**:

```typescript
{
  userId: string;            // 用户 ID
}
```

**返回数据**:

```typescript
{
  code: number;
  data: {
    id: string;
    userId: string;
    appId: string;
    createAt: string;
  };
  message?: string;
}
```

---

## 3. 移除成员

**接口**: `DELETE /api/v1/apps/:appId/members/:userId`

**使用场景**: 点击移除成员按钮，确认后调用。

**路径参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| appId | string | 是 | 应用 ID |
| userId | string | 是 | 用户 ID |

**返回数据**:

```typescript
{
  code: number;
  data: null;
  message?: string;
}
```

---

## 4. 获取成员角色配置

**接口**: `GET /api/v1/apps/:appId/members/:userId/roles`

**使用场景**: 点击成员的角色分配按钮，打开角色分配面板时加载。

**路径参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| appId | string | 是 | 应用 ID |
| userId | string | 是 | 用户 ID |

**返回数据**:

```typescript
{
  code: number;
  data: Array<{
    roleId: string;
    roleName: string;
    roleCode: string;
    isBuiltin: number;
  }>;
  message?: string;
}
```

---

## 5. 更新成员角色配置

**接口**: `PUT /api/v1/apps/:appId/members/:userId/roles`

**使用场景**: 角色分配面板保存，提交成员的角色列表。

**路径参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| appId | string | 是 | 应用 ID |
| userId | string | 是 | 用户 ID |

**请求体**:

```typescript
{
  roleIds: string[];         // 角色 ID 列表（全量替换）
}
```

**返回数据**:

```typescript
{
  code: number;
  data: {
    userId: string;
    appId: string;
    updatedCount: number;
  };
  message?: string;
}
```

---

## 6. 获取可选角色列表

**接口**: `GET /api/v1/apps/:appId/members/available-roles`

**使用场景**: 角色分配面板加载，展示可选角色（内置角色 + 应用级角色）。

**路径参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| appId | string | 是 | 应用 ID |

**返回数据**:

```typescript
{
  code: number;
  data: Array<{
    id: string;
    roleName: string;
    roleCode: string;
    isBuiltin: number;
    isOwner: number;       // 拥有者角色不可分配
  }>;
  message?: string;
}
```
