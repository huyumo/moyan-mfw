# 数据模型

## 概述

使用 TypeORM 定义实体，MySQL 8.0 存储，所有业务实体继承 `Base` 基类，支持软删除。

## 基类 Base

| 字段 | 类型 | 说明 |
|------|------|------|
| createdAt | datetime | 创建时间（自动） |
| updateAt | datetime | 更新时间（自动） |
| deleteAt | datetime | 软删除时间（null=未删除） |

## 实体关系

```
┌──────────┐     1:N     ┌──────────────┐
│ AppType  │────────────→│AppTypePermission│ 权限池
└──────────┘             └──────────────┘
     │ 1:N                      │ N:1
     ↓                          ↓
┌──────────┐             ┌──────────────┐
│   App    │             │  Permission  │
└──────────┘             └──────────────┘
     │ 1:N                      ↑ N:1
     ↓                          │
┌──────────┐             ┌──────────────────┐
│AppMember │             │RolePermission    │
└──────────┘             └──────────────────┘
     │ N:1        N:1 ↑          ↑ N:1
     ↓                │          │
┌──────────┐    ┌──────────┐  ┌──────────┐
│   User   │    │UserRole  │  │   Role   │
└──────────┘    └──────────┘  └──────────┘
                                  │
                            N:1   ↓
                           ┌──────────┐
                           │AppType   │
                           │App       │
                           └──────────┘
```

## sys_users - 用户表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 用户ID |
| username | varchar(64) | UNIQUE, NOT NULL | 登录账号 |
| password | varchar(255) | NOT NULL | bcrypt 密码哈希 |
| nickname | varchar(64) | nullable | 昵称 |
| phone | varchar(20) | UNIQUE, nullable | 手机号 |
| email | varchar(128) | nullable | 邮箱 |
| avatar | varchar(255) | nullable | 头像URL |
| gender | tinyint | default 0 | 性别（0:未知 1:男 2:女） |
| userStatus | tinyint | INDEX, default 1 | 状态（1:启用 0:禁用） |
| isDeveloper | tinyint | default 0 | 开发者标记（拥有全部权限） |

## sys_roles - 角色表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 角色ID |
| appId | char(36) | INDEX, nullable | 应用实例ID（空=全局角色） |
| appTypeId | char(36) | INDEX, nullable | 应用类型ID（空=全局角色） |
| roleName | varchar(64) | NOT NULL | 角色名称 |
| roleCode | varchar(64) | UNIQUE | 角色编码 |
| roleDesc | varchar(255) | nullable | 描述 |
| isBuiltin | tinyint | default 0 | 内置角色（不允许删除） |
| isOwner | tinyint | INDEX, default 0 | 拥有者角色（拥有全部权限） |
| roleStatus | tinyint | INDEX, default 1 | 状态（1:启用 0:禁用） |
| sortOrder | int | default 0 | 排序号 |

## sys_user_roles - 用户角色关联表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| userId | char(36) | INDEX, FK → User | 用户ID |
| roleId | char(36) | INDEX, FK → Role | 角色ID |

联合唯一约束：(userId, roleId)

## sys_permissions - 权限表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 权限ID |
| permName | varchar(64) | NOT NULL | 权限名称 |
| permCode | varchar(128) | UNIQUE | 权限编码（按树结构自动组装） |
| permDesc | varchar(255) | nullable | 描述 |
| permissionType | enum(PC/NORMAL) | INDEX | 权限类型 |
| nodeType | enum(MENU/PAGE/TAG) | nullable | 节点类型 |
| parentId | char(36) | INDEX, nullable, FK | 父权限ID |
| routePath | varchar(255) | INDEX, nullable | 前端路由路径 |
| isAutoSync | tinyint | INDEX, default 0 | 是否自动同步生成 |
| externalUrl | varchar(500) | nullable | 外部链接 |
| iconName | varchar(64) | nullable | 图标名称 |
| sortOrder | int | default 0 | 排序号 |
| isVisible | tinyint | default 1 | 是否在菜单中显示 |
| isCache | tinyint | default 1 | 是否缓存页面 |
| showMode | enum(NORMAL/DEV) | default NORMAL | 显示模式 |
| permStatus | tinyint | INDEX, default 1 | 状态（1:启用 0:禁用） |
| permissionValue | bigint unsigned | default 0 | 位运算权限值 |

支持自引用树结构：parent → children (OneToMany)

## sys_role_permissions - 角色权限关联表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| roleId | char(36) | INDEX, FK → Role | 角色ID |
| permissionId | char(36) | INDEX, FK → Permission | 权限ID |
| permissionValue | bigint unsigned | NOT NULL | 位运算权限值（细粒度） |

联合唯一约束：(roleId, permissionId)

## sys_app_types - 应用类型表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 应用类型ID |
| typeName | varchar(64) | NOT NULL | 类型名称 |
| typeCode | varchar(64) | UNIQUE | 类型编码 |
| typeDesc | varchar(255) | nullable | 描述 |
| icon | varchar(128) | nullable | 图标 |
| multiAppEnabled | tinyint | default 0 | 是否支持多应用 |
| typeStatus | tinyint | INDEX, default 1 | 状态（1:启用 0:禁用） |
| sortOrder | int | default 0 | 排序号 |

## sys_app_type_permissions - 应用类型权限池表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| appTypeId | char(36) | INDEX, FK → AppType (CASCADE) | 应用类型ID |
| permissionId | char(36) | INDEX, FK → Permission (CASCADE) | 权限ID |
| permissionValue | bigint unsigned | default 0 | 权限值（Permission.permissionValue子集） |

联合唯一约束：(appTypeId, permissionId)

## sys_apps - 应用实例表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 应用实例ID |
| appName | varchar(128) | NOT NULL | 应用名称 |
| appCode | varchar(64) | UNIQUE | 应用编码 |
| appDesc | varchar(512) | nullable | 应用描述 |
| appTypeId | char(36) | INDEX, nullable | 应用类型ID |
| ownerId | char(36) | INDEX | 负责人用户ID |
| icon | varchar(255) | nullable | 应用图标 |
| appStatus | tinyint | INDEX, default 1 | 状态（1:启用 0:禁用） |
| sortOrder | int | default 0 | 排序号 |

## sys_app_members - 应用成员表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| appId | char(36) | INDEX, FK → App | 应用ID |
| userId | char(36) | INDEX, FK → User | 用户ID |

联合唯一约束：(appId, userId)

## sys_audit_logs - 审计日志表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 日志ID |
| module | enum(AUTH/USER/ROLE/PERMISSION/APP/APP_TYPE/MEMBER/SYSTEM) | INDEX | 所属模块 |
| event | varchar(64) | INDEX | 事件名称 |
| operatorId | char(36) | INDEX | 操作人ID |
| operatorName | varchar(64) | NOT NULL | 操作人名称 |
| targetId | char(36) | INDEX, nullable | 目标ID |
| targetType | varchar(64) | NOT NULL | 目标类型 |
| description | text | NOT NULL | 操作描述 |
| snapshot | json | nullable | 数据快照（before/after） |
| ip | varchar(64) | NOT NULL | IP地址 |
| userAgent | text | nullable | User-Agent |
| createAt | datetime | INDEX, 自动 | 创建时间 |
