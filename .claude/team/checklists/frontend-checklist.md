# 前端工程师角色检查清单

## 组件设计

- [ ] 组件职责是否单一
- [ ] Props 定义是否清晰
- [ ] 事件命名是否规范
- [ ] 状态管理是否合理
- [ ] 是否可复用

## 用户体验

- [ ] 加载状态是否有反馈
- [ ] 错误状态是否有提示
- [ ] 空状态是否有引导
- [ ] 操作是否有确认（危险操作）
- [ ] 表单是否有验证

## 交互逻辑

- [ ] 边界条件是否处理（空数据、长文本、特殊字符）
- [ ] 异步操作是否有 loading
- [ ] 网络错误是否有重试/降级
- [ ] 权限控制是否正确

## 代码规范

- [ ] 是否符合 Vue 代码规范
- [ ] TypeScript 类型是否完整
- [ ] 是否有未使用的导入/变量
- [ ] 样式是否隔离
- [ ] 响应式依赖是否正确

## 性能优化

- [ ] 列表是否有 key
- [ ] 是否有不必要的渲染
- [ ] 大图/多图是否懒加载
- [ ] 组件是否按需加载

## 与后端接口

- [ ] 接口定义是否与文档一致
- [ ] 错误码处理是否完整
- [ ] 数据格式转换是否正确
- [ ] 是否有防抖/节流
- [ ] 接口变更是否有沟通确认

## 可访问性 (可选)

- [ ] 是否支持键盘操作
- [ ] 焦点管理是否正确
- [ ] 图片是否有 alt 文本
- [ ] 颜色对比度是否达标

## 国际化 (如适用)

- [ ] 文案是否使用 i18n
- [ ] 日期/时间格式是否正确
- [ ] 数字/货币格式是否正确
